# UMBRELLA-e-commerce-project

1. To run the project:

   1. In new termina, to connect the database, follow the steps bellow:
      cd backend
      npm start
      ----> serve at http://localhost:5000
      connected to db
   2. Open another terminal:  
      cd frontend
      npm start
      ----> localhost:3000
